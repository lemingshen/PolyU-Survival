# Zotero Connector

Zotero Connector is a browser extension that allows you to save references and PDFs from the web directly into your Zotero library. With this extension, you can easily capture bibliographic information, full-text PDFs, and web snapshots with just a click.

When you install Zotero Connector, it will add a button to your browser's toolbar. Clicking this button will save the current webpage or PDF to your Zotero library, along with any available metadata.
